## Описание

Домашнее задание к занятию «Работа с HTTP».

Реализован frontend для HelpDesk:

- отображение всех тикетов;
- создание нового тикета;
- просмотр подробного описания тикета;
- редактирование тикета;
- удаление тикета;
- изменение статуса выполнения тикета.
